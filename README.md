# GravitySim

This project is a real time gravity simulation written in C with SDL2. It's built around the idea of simulating planetary and orbital motion in a way thats grounded in actual physics.

The project focuses on bodies interacting under gravity in real time, with rendering, trails, controls, and preset scenarios to explore different kinds of motion. The long term goal is to keep pushing it toward a more accurate and more complete physics simulation rather than treating it like a game.

It's also meant to be a learning project for myself just out of curisotiy to deepen my understanding of physics. This whole project was inspired by a black hole simulation in C++. My goal was to do something similar and learn more about that field.
